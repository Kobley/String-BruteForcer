# String/Password BruteForcer
(not a dictionary attack!)

>## Python Setup
```
> git clone https://github.com/Kobley/Password-BruteForcer.git

> cd Python/

> python3 PasswordBruteForcer.py
```

>## Javascript Setup
```
> git clone https://github.com/Kobley/Password-BruteForcer.git

> cd JavaScript/

> npm i

> node ./main.js
```
